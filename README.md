### Keyword Weight Techniques

**Weight Control Methods**:

1. **Position Weight Method**
   ```
   Earlier position = Higher weight

   ‚úÖ "beautiful sunset, mountains, lake"
   (Emphasizes beautiful sunset)

   ‚úÖ "mountains and lake at beautiful sunset"
   (Emphasizes mountains and lake)
   ```

2. **Repetition Emphasis Method**
   ```
   Repetition = Increased weight

   ‚úÖ "highly detailed face, detailed eyes, detailed skin texture"
   (Multiple mentions of detail, AI will focus more on detail)
   ```

3. **Description Detail Method**
   ```
   More detailed = Higher weight

   ‚ùå "a cat"
   ‚úÖ "a fluffy Persian cat with bright blue eyes and white fur"
   (Detailed description yields more expected results)
   ```

4. **Modifier Stacking Method**
   ```
   ‚úÖ "extremely detailed, ultra realistic, high quality, 8k resolution"
   (Multiple quality modifiers used in combination)
   ```

**Weight Practical Comparison**:
```
Scenario: Generate a cat image, focus on eyes

‚ùå Low weight: "a cat with blue eyes"
üü° Medium weight: "a cat, beautiful blue eyes, detailed"
‚úÖ High weight: "a beautiful cat with striking blue eyes,

4. ‚úÖ Reasonable composition, aesthetically pleasing

```
